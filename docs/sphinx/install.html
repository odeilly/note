1.1. インストール¶

1.1.1. Pythonのインストール¶

1.1.1.1. ダウンロード¶

下記ページから, インストーラをダウンロードします.

https://www.python.org/downloads/

1.1.1.2. インストール¶

  1. インストーラを起動します.

  2. 「Add Python 3.8 to PATH」にチェックを入れます.

  3. 「Install Now」をクリックします.

  4. 「Disable path length limit」をクリックします.

  5. 「Close」をクリックし, インストーラを閉じます.

1.1.1.3. 動作確認¶

コマンドプロンプトを起動し, 以下のコマンドを実行し, バージョンが表示されればインストール成功です.

C:\>python -V
Python 3.8.3

1.1.2. Sphinxのインストール¶

1.1.2.1. インストール¶

コマンドプロンプトを起動し, 以下のコマンドを実行します.

C:\>pip install sphinx Pillow

1.1.2.2. 動作確認¶

コマンドプロンプトを起動し, 以下のコマンドを実行し, バージョンが表示されればインストール成功です.

C:\>sphinx-quickstart --version
sphinx-quickstart 3.1.1

1.1.3. Make of Windowsのインストール¶

1.1.3.1. ダウンロード¶

下記ページから, インストーラをダウンロードします.

http://gnuwin32.sourceforge.net/packages/make.htm

1.1.3.2. インストール¶

  1. インストーラを起動し, デフォルト設定のままインストールします.

  2. システム環境変数 Path に, C:\Program Files (x86)\GnuWin32\bin を追加します.

1.1.3.3. 動作確認¶

コマンドプロンプトを起動し, 以下のコマンドを実行し, バージョンが表示されればインストール成功です.

C:\>make -version
GNU Make 3.81

1.1.4. TeX Liveのインストール¶

1.1.4.1. ダウンロード¶

下記ページから, インストーラをダウンロードします.

http://www.tug.org/texlive/acquire-netinstall.html

1.1.4.2. インストール¶

インストーラを起動し, デフォルト設定のままインストールします. ただし, TeXworksは特に利用しないので, 不要の場合はチェックを外して下さい.

1.1.4.3. 動作確認¶

コマンドプロンプトを起動し, 以下のコマンドを実行し, バージョンが表示されればインストール成功です.
