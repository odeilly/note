1.2. テーマ¶

Sphinxでは, 10個の組み込みテーマがあります. また, サードパーティ製のテーマを利用することもできます.

1.2.1. bootstrapテーマを利用する¶

https://github.com/ryan-roemer/sphinx-bootstrap-theme

1.2.1.1. ダウンロード¶

コマンドプロンプトで, 下記を実行します.

C:\>pip install sphinx_bootstrap_theme

1.2.1.2. 適用¶

conf.py に, 以下を追加します.

# At the top.
import sphinx_bootstrap_theme

#...

# Activate the theme.
html_theme = 'bootstrap'
html_theme_path = sphinx_bootstrap_theme.get_html_theme_path()

1.2.2. Read the Docs テーマを利用する¶

https://sphinx-rtd-theme.readthedocs.io/en/stable/

1.2.2.1. ダウンロード¶

コマンドプロンプトで, 下記を実行します.

pip install sphinx-rtd-theme

1.2.2.2. 適用¶

conf.py に, 以下を追加します.

import sphinx_rtd_theme

extensions = [

    ...

    'sphinx_rtd_theme',

    ...

]

html_theme = 'sphinx_rtd_theme'

1.2.3. Bootstrap v4 HTML Theme for Sphinx を利用する¶

https://github.com/myyasuda/sphinxbootstrap4theme

1.2.3.1. ダウンロード¶

コマンドプロンプトで, 下記を実行します.

pip install sphinxbootstrap4theme

1.2.3.2. 適用¶

import sphinxbootstrap4theme

html_theme = 'sphinxbootstrap4theme'
html_theme_path = [sphinxbootstrap4theme.get_path()]
